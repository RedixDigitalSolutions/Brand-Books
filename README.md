# Modern Brand Books Library (2025 Edition)

A responsive web application that displays a curated collection of 20 modern brand guidelines and corporate identity manuals from 2024 and 2025. The project is built with semantic HTML5, modern CSS3, and vanilla JavaScript, loading data dynamically from a JSON file.
